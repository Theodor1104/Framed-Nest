# Framed Nest - Design Prompts for Leonardo.ai

> Prompts til at generere plakat-designs til Framed Nest
> Platform: https://app.leonardo.ai/

---

## Indstillinger

| Setting | Anbefalet |
|---------|-----------|
| Model | Leonardo Diffusion XL eller Phoenix |
| Aspect Ratio | 2:3 (portrait - plakat-format) |
| Guidance Scale | 7-8 |
| Steps | 30-40 |
| Download | Højeste opløsning (mindst 4000px) |

**Negative prompt (brug altid):**
```
text, watermark, signature, blurry, low quality, distorted, deformed, ugly, bad anatomy
```

## Tips

1. **Undgå i prompts:** "4k", "ultra realistic", "photorealistic" - holder det mere kunstnerisk
2. **Tilføj altid:** "print quality", "gallery art", "museum quality"
3. **Test:** Generer 4 variationer og vælg den bedste
4. **Efterbehandling:** Brug Canva eller Photoshop til at justere kontrast/farver hvis nødvendigt
